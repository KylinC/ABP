ATC-BigData-Platform

基于 python Flask + Jquery + Echarts 的前端可视化框架，支持点击事件，支持实时Neo4j数据交互。用于数据挖掘与知识实体抽取的可视化平台。

DEpendencies






项目前端依赖 Python Flask 框架开发，依赖 Jquery 事件响应、Echarts页面渲染，后端依赖Python实时计算（如用Jieba做分词、用Sklearn构建支持向量机）或调用局域网内的Neo4j数据库。

简介

本项目主要建立一个空管知识图谱前端平台的研发以及知识图谱知识嵌入领域的研究。其空管知识图谱平台建成后将具备的功能有数据库实施提取，实时计算，系统交互，预测延误，预测感染机场的功能。平台主要应用知识图谱neo4j作为后台数据库，利用先有的流控报文，机场信息，气象报文等信息，建立一个空管领域的知识图谱。另外使用机器学习方法，网络模型等实现延误预测以及标记感染机场的功能。
